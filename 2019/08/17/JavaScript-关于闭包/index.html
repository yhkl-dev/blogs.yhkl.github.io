1. 闭包的概念

闭包指的是： 能够访问另一个函数作用域得变量的函数

代码实例：

代码流程：

  • 定义函数A
  • 在A中定义函数B，并将B返回
  • 执行A， 并把A的返回结果赋值给C
  • 执行C

总结下来就是刚才开头所说的： 函数A的内部函数B被函数A外的变量C引用

这就是闭包

2. 闭包的作用

JavaScript中的GC机制和Python中的GC机制类似， 都是如果一个对象不再被引用， 那么这个对象就会被GC回收，否则这个对象会一直保存在内存中。

在上述例子中， B在A中定义， 因此B依赖A， 而外部变量C又引用了B， 所以A间接的被C引用， 所以A不会被GC回收， 会一直保存在内存中。
