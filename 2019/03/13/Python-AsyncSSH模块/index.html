Python AsyncSSH模块

AsyncSSH是一个Python实现异步ssh的模块，基于Python3.4 + asyncio框架之上提供SSHv2协议的异步客户端和服务器实现

功能

  • 完全支持SSHv2，SFTP和SCP客户端和服务器功能

    • Shell，命令和子系统通道
    • 环境变量，终端类型和窗口大小
    • 直接和转发的TCP / IP通道
    • OpenSSH兼容的直接和转发的UNIX域套接字通道
    • 本地和远程TCP / IP端口转发
    • 本地和远程UNIX域套接字转发
    • 通过SOCKS转发动态TCP / IP端口
    • 客户端和服务器上的X11转发支持
    • 具有OpenSSH扩展的SFTP协议版本3
    • SCP协议支持，包括第三方远程到远程拷贝

  • 单个SSH连接上的多个同时会话

  • 单个事件循环中的多个SSH连接

  • 具有可设置编码的字节和基于字符串的I / O.

  • 各种密钥交换，加密和MAC算法

  • 支持

    gzip压缩

    • 包括OpenSSH变体以延迟压缩直到auth之后

  • 基于用户和主机的公钥，密码和键盘交互式身份验证方法

  • 许多类型和格式的

    公钥和证书

    • 包括对RFC 6187中定义的X.509证书的支持

  • 支持在UNIX系统上访问由

    ssh-agent

    管理的密钥

    • 在客户端和服务器上都包含代理转发支持

  • 支持在Windows上访问由PuTTY的Pageant代理管理的密钥

  • 支持通过OpenSSH的ssh-keysign访问主机密钥

  • OpenSSH风格的known_hosts文件支持

  • OpenSSH风格的authorized_keys文件支持

  • 兼容OpenSSH“加密然后MAC”选项以提高安全性

  • 基于时间和字节数的会话密钥重新协商

  • 旨在易于扩展，以支持新形式的密钥交换，身份验证，加密和压缩算法
